## Project Overview

This is a simple recipes website built with HTML. The project demonstrates foundational HTML skills by creating a structured, multi-page recipe page where users can browse different recipes and their ingredients.

## Skills Demonstrated

By completing this project, you will have demonstrated proficiency with the following HTML elements:

- **Headings** (`h1-h6`): Structuring content with appropriate heading hierarchy
- **Paragraphs** (`p`): Organizing text content
- **Images** (`img`): Embedding recipe photos and visual elements
- **Links** (`a`): Navigation between recipe pages
- **Unordered Lists** (`ul`) and **List Items** (`li`): Displaying ingredients
- **Ordered Lists** (`ol`) and **List Items** (`li`): Showing step-by-step cooking instructions
